Silent Security

Two part, bluetooth connected device. Designed for weary travelers, the motion sensor is placed inside the luggage, while the user holds the second piece. If the motion sensor senses movement (done through a gyroscope), the second piece will vibrate uncontrollably (thorugh a motor) until the user places their fingerprint on the sensor.

Materials: Arduino, Gyroscope, Bluetooth, Fingerprint Module

Zombie Dash

Multi-level game where the player navigates through a map to avoid hitting zombies and rescuing civilians. The player can use flamethrowers and land mines to extinguish zombies, but must also ensure that the civilians get to the exit safely.

Languages: C++

Night at the Diner

Highly interactive escape room game where the player must find clues hidden throughout the room and escape. Blender and Illustrator were used alongside WebGL/tiny graphics library to truly immerse the player into the experience.

Languages: JavaScript, WebGL

Technologies Used: Blender, Adobe Illustrator

Tasks

Simple CRUD Android application where the user and add, edit, and remove tasks.

Languages: Kotlin

Technologies: SQLite Database

OPS Capstone 2019

Radio controlled car that moves based on the actions of an external controller, linked through a radio module. Simple PCBs designed using EAGLE are attached to the controller and the car.

Languages: C

Technologies: Arduino, Radio Module, Gyroscope, EAGLE
